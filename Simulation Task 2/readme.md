This readme file is for the instruction to run the script properly.

These scripts are developed on Python 3 enviroment.
Please install the require package by the following command.

pip3 install -r requirement.txt.

Once the package install, to run the task 2.1, run the command as following:

python3 simulation_task2_1.py

For the task2.2, do the following:

python3 simulation_task2_1.py
